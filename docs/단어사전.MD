explicitly 부사 (→implicitly)

explicit  ( 부사형: explicitly ) 
1.형용사 분명한, 명쾌한
2.형용사 솔직한, 터놓고 말하는 (=frank)
3.형용사 명백한, 노골적인 (→implicit)

corresponding  
형용사 (~에) 해당[상응]하는 (=equivalent)

satisfy   
1.동사 만족시키다
2.동사 (필요·욕구 등을) 충족시키다[채우다]
3.동사 확신하게 하다, 납득시키다

various   
1.형용사 여러 가지의, 각양각색의, 다양한 (=diverse)
2.형용사 격식 다양한 특징을 지닌 (=diverse)
